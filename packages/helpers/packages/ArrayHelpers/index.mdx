<a name="arrayItemAddProp"></a>

## arrayItemAddProp
adds similar properties to children of an array

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| baseArray | array | the given array to add property to items |
| props | object | object that should be attached to each item |

<a name="flattenArray"></a>

## flattenArray
makes a flat array from sub-arrays of an array

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| array | array | given array to flatten sub-objects |
| property | string | name of the key of the of the sub-objects that want to be merged |

<a name="stringifyArray"></a>

## stringifyArray
Create string from array

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| array | array | base array that wants to be stringify |
| properties | array | list of properties that want to be in stringified result |

<a name="deepFlatten"></a>

## deepFlatten
Deep flatten an array

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| array | array | the multi dimensional array that want to be flatten |
| property | string | name of the property of array that want to navigate through |
| levelKey | string | name of the property you want to store the level of item |

<a name="arraySeparator"></a>

## arraySeparator
separates an array based on criterion into several groups

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| array | array | array of strings |
| separators | object | object of regex separators |
| noDuplicates | boolean | flag to remove duplicates |
