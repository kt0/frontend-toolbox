## get started
We provide two way of using this package `single` or `multi` :
```bash
npm i @snappmarket/number-helpers
OR
npm i @snappmarket/helpers
```

## persianNumber
converts the english number to persian number

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| string | string | the string that contains the number  you want to convert to persian |

<a name="zeroPadding"></a>

## zeroPadding
adds starting zeros to numbers

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| number | number | number you want to add zero to its start |
| padding | number | count of zeros you want to add for a number to fix its length |
