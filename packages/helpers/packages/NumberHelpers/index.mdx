## numericInputValidation
**Kind**: global function

check input value,then is number or not. and check min and max input value integer
you can create a input by "tel" type and set onChenge for it,
in onChange, pass min,max and inputValue to the numericInputValidation function and use that result;

``` es6
const { value, isValid, message } = numericInputValidation({
  inputValue,
  min,
  max,
  validationMessages: {
    minMessage: {
      type: 'danger',
      content: 'min',
    },
    maxMessage: {
      type: 'danger',
      content: 'max',
    },
    nanMessage: {
      type: 'danger',
      content: 'nan',
    },
  }
});
```

| Param | Type | Description |
| --- | --- | --- |
| min | number | the english number |
| max | number | the english number |
| inputValue | string | the english number string like "1000" |
| validationMessages | shape | string for all like top example|
