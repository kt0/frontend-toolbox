## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useResizeObserver } from '@snappmarket/hooks';

const SampleComponent = props => {
  const { height, width, left, top } = useResizeObserver(ref);

  /**
  * rest of component code
  */
};
```
